Um pouco sobre Front-end



Um pouco das ferramentas de um desenvolvedor front-end

Basicamente as ferramentas de um programador front-end são:

1- [VS code - O VS code é uma ferramenta muito usada no front-end pois, ela é uma editora de codigos fonte leve, mas muito potente, tendo uma ligação com o Github por meio do code space.]

2- [Github - É um serviço de hospedagem de codigosna nuvem, alem de ser bem parecido com uma rede social, já que você pode deixar seu codigo a mostra para outros devs verem o mesmo e te ajudar caso haja algum problema com o mesmo.]

3- [Git - O Git por sua vez é usado como um é usado para gerenciar e versionar seu codigo, assim, caso voçê precise modificar algo em uma parte do seu codigo, voçê não precise modificar a linha de codigo já pronta, assim não gerando futuras complicações.]

4- [Bitbucket - O Bitbucket assim como o Github também é um serviço de hospedagem de codigo, sendo comumente usado por empresas, alem de que ele utiliza da linguem de phyton.]



Versionamento e sua importancia

O versionamento tem uma importancia gigante, por conta do risco de modificar um codigo importante e gerar um conflito caso o novo codigo, mas também, o versionamento é importante para assegurar o dev de tenha total certeza e segurança do que ele esta fazendo já que, com o aumento da procura por desenvolvedores front e back-end se tornou vital o segurança do codigo.

Basicamente o desenvolvedor de front-end é responsável pelo rosto, corpo, conteudo e visual de uma pagina ou aplicativo
