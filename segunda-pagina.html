Um pouco sobre Front-end

O front-end é O que é Front End? Podemos classificar como a parte visual de um site, aquilo que conseguimos interagir. Quem trabalha com Front End é responsável por desenvolver por meio do código uma interface gráfica e, normalmente, com as tecnologias base da Web: HTML, CSS e JavaScript.

Alem de que o front-end ser o "rosto do site" ele é a porta de abertura para o back-end, sendo ele muito importante para a criação de um site ou aplicativo. Normalmente o front-end é retratado como a forma de criação mais comum tendo varias ferramentas utilizaveis como o HTML, CSS e JavaScript.

Os desenvolvedores front-end tem um trabalho arduo em suas mãos sendo ele a opção de tornar seu site mais inclusivo para pessoas com deficiencias visuais, tornar o codigo mais simples para facilitar a leitura por outros desenvolvedores e por leitores de tela.


Um pouco das ferramentas de um desenvolvedor front-end

Basicamente as ferramentas de um programador front-end são:

1- [VS code - O VS code é uma ferramenta muito usada no front-end pois, ela é uma editora de codigos fonte leve, mas muito potente, tendo uma ligação com o Github por meio do code space.] 2- [Github - É um serviço de hospedagem de codigosna nuvem, alem de ser bem parecido com uma rede social, já que você pode deixar seu codigo a mostra para outros devs verem o mesmo e te ajudar caso haja algum problema com o mesmo.] 3- [Git - O Git por sua vez é usado como um é usado para gerenciar e versionar seu codigo, assim, caso voçê precise modificar algo em uma parte do seu codigo, voçê não precise modificar a linha de codigo já pronta, assim não gerando futuras complicações.] 4- [Bitbucket - O Bitbucket assim como o Github também é um serviço de hospedagem de codigo, sendo comumente usado por empresas, alem de que ele utiliza da linguem de phyton.]
Vercionamento e sua importancia

O versionamento tem uma importancia gigante, por conta do risco de modificar um codigo importante e gerar um conflito caso o novo codigo, mas também, o versionamento é importante para assegurar o dev de tenha total certeza e segurança do que ele esta fazendo já que, com o aumento da procura por desenvolvedores front e back-end se tornou vital o segurança do codigo.
